# 🚀 실행 컨텍스트 🚀

- 실행할 코드에 관한 환경 정보들을 모아놓은 객체
- scope, hoisting, this, closure 등의 개념을 포함

<br>

# 실행 컨텍스트가 활성화 되는 시점

- 호이스팅이 발생한다.(선언된 변수를 위로 끌어올린다.)
- 외부 환경 정보를 구성한다.
- this값을 설정한다

<br>

```javascript
var a = 1; // 전역 컨텍스트
function outer() {
  // outer 컨텍스트
  function inner() {
    // inner 컨텍스트
    console.log(a); // undefined
    var a = 3;
    console.log(a); // 3
  }
  inner();
  console.log(a); // 1
}
outer();
console.log(a); // 1
```

### 실행 컨텍스트가 구성되는 코드들은 다음과 같다

- 전역 코드
- eval 코드
- 함수 코드
- 모듈 코드
